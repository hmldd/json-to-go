JSON-to-Go

JSON转化为Go结构体

JSON-to-Go工具可以实时将JSON转化为Go结构体。左边粘贴JSON格式的字符串右边自动生成可直接在项目中使用的Go结构体。JSON-to-Go工具有时需要做一些假设，请再检查一遍输出结果！

试试转化JSON SmartyStreets API 或 GitHub API 吧。
